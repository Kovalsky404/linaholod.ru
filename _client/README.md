# \_client

Рабочая папка с материалами клиента.

- **brief.md** — бриф проекта (под git).
- **references/** — референсы и заметки (под git).
- **assets/** — медиа клиента: фото, логотипы (НЕ под git).
- **data/** — тексты, прайсы, контент (НЕ под git).

`assets/` и `data/` исключены из репозитория в `.gitignore`, чтобы не
коммитить тяжёлые/приватные материалы. Храните их локально или в облаке.
